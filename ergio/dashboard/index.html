🤖 AI Agent Frameworks

        
Connect ERGIO to external AI agent systems for enhanced capabilities

      


      

        
How Agent MCP Works

        
ERGIO's Conductor acts as an MCP Client — it connects to external AI agent frameworks via the Model Context Protocol. When you give ERGIO a task, it can delegate to specialized agents (like Hermes or OpenClaw) that have their own memory, tools, and intelligence. These agents get better over time — they remember what worked and create new skills from successful interactions.
